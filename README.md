# Simple Chat App with CRUD 

A beginner-friendly chat web app where users can send messages and view:
- **Sender name**
- **Receiver name**
- **Message date**
- Full **CRUD operations** (Create, Read, Update, Delete)

##  Features
- Send and display chat messages
- Shows sender & receiver names
- Displays message date
- Edit messages (Update)
- Delete messages
- Built while learning backend + database + REST APIs

##  Tech Used
- Node.js
- Express.js
- EJS (templating)
- MongoDB / Mongoose
- REST API (CRUD)

##  CRUD Operations Supported
| Method | Operation |
|--------|-----------|
| POST   | Create new chat |
| GET    | Read / show chats |
| PATCH  | Update chat message |
| DELETE | Delete chat |

##  Learning Purpose
This project was created to practice **database operations and RESTful routing**.  
Perfect for beginners exploring backend development.
